# 1 | SQL (Search Query Language)
## 1.1 | Pengertian Awal
Bahasa standar yang digunakan untuk mengelola dan memanipulasi database relasional. Dimana keguaanyanya meliputi:
- INSERT
- SELECT
- UPDATE
- DELETE
- CREATE, ALTER, DROP
- GRANT, REVOKE

> Source: Materi kuliah

SQL (Structured Query Language) adalah bahasa standar yang digunakan untuk mengelola dan berinteraksi dengan basis data relasional. Bahasa ini memungkinkan pengguna untuk mengajukan pertanyaan (query), menambah, memperbarui, menghapus, dan mengatur data di dalam basis data. SQL sangat penting dalam berbagai industri dan karier terkait data, seperti ilmu data dan analisis data.

Bahasa kueri terstruktur (SQL) adalah bahasa pemrograman untuk menyimpan dan memproses informasi dalam basis data relasional. Basis data relasional menyimpan informasi dalam bentuk tabel, dengan baris dan kolom yang mewakili berbagai atribut data dan berbagai hubungan antar nilai data. Anda dapat menggunakan pernyataan SQL untuk menyimpan, memperbarui, menghapus, mencari, dan mengambil informasi dari basis data. Anda juga dapat menggunakan SQL untuk memelihara dan mengoptimalkan kinerja basis data.

> Source: [What is SQL? - Structured Query Language (SQL) Explained - AWS](https://aws.amazon.com/what-is/sql/)
## 1.2 | Mengapa SQL

1. **Standar:** semua databse relasional mendukung SQL.

2. **Efisiensi:** manipulasi data besar secara efisien dengan operasi sederhana.

3. **Portabilitas:** dapat digunakan berbagai system database,

4. **Fleksibilitas:** query sederhana hingga kompleks.

> Source: Materi kuliah

Bahasa kueri terstruktur (SQL) adalah bahasa kueri populer yang sering digunakan di semua jenis aplikasi. Analis dan pengembang data mempelajari dan menggunakan SQL karena terintegrasi dengan baik dengan berbagai bahasa pemrograman. Misalnya, mereka dapat menyematkan kueri SQL dengan bahasa pemrograman Java untuk membangun aplikasi pemrosesan data berkinerja tinggi dengan sistem basis data SQL utama seperti Oracle atau MS SQL Server. SQL juga cukup mudah dipelajari karena menggunakan kata kunci umum dalam bahasa Inggris dalam pernyataannya.

> Source: [What is SQL? - Structured Query Language (SQL) Explained - AWS](https://aws.amazon.com/what-is/sql/)

## 1.3 | Fungsi utama SQL

1. **Mengelola data:** SQL digunakan untuk membuat, mengambil, memperbarui, dan menghapus data dalam sebuah basis data. 

2. **Mengelola struktur basis data:** Selain data, SQL juga dapat digunakan untuk membuat dan memodifikasi tabel serta struktur basis data lainnya. 
    
3. **Mengakses dan menganalisis data:** Memungkinkan pengguna untuk mengajukan pertanyaan kompleks tentang data, menganalisisnya, dan membuat laporan. 

4. **Memungkinkan analisis yang canggih:** Pengguna dapat menggabungkan data dari beberapa tabel dan melakukan agregasi data tanpa harus merinci cara pengambilan data tersebut. 

> Source: [SQL Adalah Structured Query Language: Ini Definisi, dan Manfaatnya - Jobstreet Indonesia](https://id.jobstreet.com/id/career-advice/article/sql-adalah#:~:text=fitur%20yang%20kompleks.-,Kesimpulan%20tentang%20apa%20itu%20SQL,atau%20Apple%20App%20Store%2C%20gratis!)

Fungsi SQL adalah untuk mengelola dan memanipulasi data di dalam database relasional dengan melakukan operasi seperti mengambil (SELECT), menambah (INSERT), memperbarui (UPDATE), dan menghapus (DELETE) data. Selain itu, SQL juga digunakan untuk mendefinisikan struktur database, seperti membuat dan menghapus tabel (CREATE TABLE, DROP TABLE), serta mengelola hak akses pengguna melalui perintah DCL (Data Control Language) seperti GRANT dan REVOKE. 

Fungsi utama SQL:

1. **Manipulasi data:** Memungkinkan Anda untuk memasukkan data baru (INSERT), mengubah data yang sudah ada (UPDATE), dan menghapus data (DELETE). 
    

2. **Pengambilan data:** Menggunakan perintah SELECT untuk mengambil data dari satu atau lebih tabel, yang bisa disaring dan diurutkan sesuai kebutuhan. 
    

3. **Manajemen struktur data:** Memungkinkan pembuatan objek database seperti tabel (CREATE TABLE), penghapusan objek (DROP TABLE), dan modifikasi struktur tabel (ALTER TABLE). 

4. **Kontrol akses:** Mengatur hak akses pengguna untuk membatasi siapa saja yang bisa melihat atau memodifikasi data (GRANT dan REVOKE). 
    
5. **Eksekusi kueri:** Menjalankan berbagai kueri untuk mengelola dan menginstruksikan sistem database. 

6. **Integrasi:** Dapat diintegrasikan dengan bahasa pemrograman lain seperti Python atau Java untuk pengembangan aplikasi yang lebih kompleks.
